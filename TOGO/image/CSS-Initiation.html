Un Flappy Bird-like dans Android

Cela fait plusieurs années qu’Android cache un mini-jeu qui reprend le principe de Flappy Bird. Android 6.0 Marshmallow le cache toujours : pour le débloquer, rendez-vous dans les paramètres du terminal, puis dans « A propos du téléphone ». Là, appuyez plusieurs fois sur la version d’Android pour faire apparaître un logo de Marshmallow. En appuyant quelques secondes dessus, le jeu se déclenche ! A noter que ce petit jeu n’est pas toujours accessible, notamment sur certains terminaux dotés de surcouches logicielles. Vous aurez plus de chance d’y accéder depuis un Nexus, ou tout autre smartphone à l’Android « pur »… Mais ne désespérez pas avant d’avoir essayé. Nous avons réussi à y accéder sur un Galaxy S6 Edge, de Samsung, fourni avec une surcouche.

Le dinosaure sauteur de Chrome

Ce jeu est caché dans toutes les versions de Chrome, que ce soit sur un ordinateur de bureau ou sur les versions mobiles. Mais pour y jouer, il faut être hors ligne : lorsque vous utilisez Chrome sans connexion Internet, un petit dinosaure apparaît avec la mention « page inaccessible ». Sur mobile, si vous appuyez sur le dino, celui-ci va lancer un mini-jeu où il vous faudra sauter par-dessus des cactus, puis éviter des oiseaux. Pour le lancer sur la version bureau de Chrome, appuyez sur la touche espace. Attention, le navigateur ne sauvegarde pas les scores, et dès qu’il trouve du réseau, il recharge la page, faisant disparaître le jeu !
